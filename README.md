 Website Profil Sekolah & PPDB Online


Aplikasi web berbasis framework modern untuk mengelola profil instansi sekolah sekaligus sistem pendaftaran siswa baru (PPDB) secara mandiri, transparan, dan efisien.

---

 Fitur Utama

 Profil Sekolah (Frontend)
* **Beranda Dinamis:** Menampilkan carousel banner terbaru, sambutan kepala sekolah, dan visi-misi.
* **Galeri Kegiatan:** Dokumentasi foto dan video fasilitas serta prestasi sekolah.
* **Berita & Artikel:** Informasi terupdate seputar kegiatan sekolah dan pengumuman penting.
* **Halaman Kontak:** Integrasi Google Maps dan formulir pesan langsung.

 Sistem PPDB Online
* **Formulir Pendaftaran:** Input data calon siswa, orang tua, dan riwayat nilai/prestasi.
* **Upload Berkas:** Unggah dokumen kelengkapan (KK, Akta Kelahiran, Ijazah/SKL).
* **Cetak Kartu Ujian:** Dokumen otomatis berformat PDF untuk bukti pendaftaran.
* **Pengumuman Seleksi:** Halaman hasil seleksi yang dapat dicek berdasarkan NISN atau nomor pendaftaran.

Panel Admin (Backend)
* **Dashboard Statistik:** Grafik jumlah pendaftar harian dan status kelulusan.
* **Manajemen Konten (CMS):** Edit informasi profil, berita, dan galeri tanpa coding.
* **Verifikasi Data PPDB:** Validasi berkas pendaftar dan penentuan status kelulusan (Diterima/Ditolak).

---

Teknologi yang Digunakan

* **Frontend:** HTML5, CSS3, JavaScript, [Bootstrap 5 / Tailwind CSS]
* **Backend:** [PHP Native / Laravel / Python Flask / Node.js]
* **Database:** MySQL
* **Tools pendukung:** Composer, Git, Laragon / XAMPP
